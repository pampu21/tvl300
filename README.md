After Cloning

# Frontend Setup
 1. Got to front end directory `tlv300-fe`
 2. run `npm install`
 3. run `cp .env.example .env`
 4. run `npm run dev`
 5. visit `http://localhost:3000`

# BackEnd Setup
 1. Got to front end directory `tlv300-be`
 2. run `npm install`
 3. run `cp .env.example .env`
 4. run `npm run start:dev`
